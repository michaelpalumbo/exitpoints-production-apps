# exitpoints-production-apps

## How to install each app

For example, the *images* program
1. run ```cd images```
2. run ```npm install```
3. run ```cd ..```
4. try running it by placing some images into the /images/images folder
5. run ```npm run images```
6. check output for letterboxed images

## Special: For the images and video program:

Please be sure to rename images and videos with the artist's name, followed by a number suffix ie:

"michael 1.jpg"
"michael 2.jpg"

## video preparation process:

After downloading the media files, organize them into the /videos/toLetterbox folder

ensure any videos that are in .mov format are converted to .mp4 using https://www.freeconvert.com/video-converter
